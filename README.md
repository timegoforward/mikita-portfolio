# Mikita Yemialyanau Portfolio

Free stack: Next.js + GitHub + Vercel.

## Local run

```bash
npm install
npm run dev
```

## Deploy

1. Create a new GitHub repository.
2. Upload this project.
3. Import the repository in Vercel.
4. Keep the default Next.js settings and deploy.

The Brand & Marketing and Product links are placeholders in `app/page.js`.

Important: the portrait currently uses a temporary Figma asset URL. Export the portrait node as PNG from Figma, save it as `public/portrait.png`, and replace the image `src` in `app/page.js` with `/portrait.png` before long-term deployment.
